# Awesome Welcome

A simple GTK3 welcome application for **Manjaro Awesome Respin**.  
Provides quick access to system setup tools,Calamares, Pamac, and Nordic wallpapers.

## Features

- Detects whether running in **Live ISO** or **installed system**.
- Launches **Calamares installer** in live environment.
- Install **Nordic wallpapers** on installed systems.
- Quick access to **Pamac software manager**.
- Autostart management for convenience.
